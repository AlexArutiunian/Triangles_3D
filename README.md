# Triangle_3D

Input: N - number of triangles and N ones of triangles in 3D.

Output: numbers of triangles that intersect with any other. 

Input:
``` bash
6
1 0 0 0 1 0 0 0 1
3 0 0 0 3 0 0 0 3
0 0 0 0 0 4 0 4 0
0 0 2 4 1 0 1 4 0
0 0 10 3 7 0 7 3 0
0 -4 0 0 -6 0 2 -5 0

```
Output:
```bash
3 4
```
Or output:
```bash
4 3
```
P.S (4 3 also is right)
